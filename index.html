The CSS Spot

CSS Tags

Property Name Description
color Text Color Sets the color of the text inside an element.
background-colors Background Color Sets the color of the text inside an element.
font-size Font Size Controls the size of the text within an element.
font-family Font Family Specifies the font style for the text.
font-weight Font Weight Sets the thickness or boldness of the font.
text-align Text Alignment Aligns the text within an element (e.g., left, right, center).
margin Margin Defines the space outside of an element.
padding Padding Specifies the space inside an element, between the content and the border.
border Border Sets the style, color, and width of an element’s border.
width Width Controls the width of an element.
height Height Sets the height of an element.
display Display Defines the display behavior of an element (e.g., block, inline, none).
position Position Specifies the positioning method of an element (e.g., static, relative, absolute, fixed, sticky).
top, right, bottom, left Position Offsets Moves an element from its default position based on its position property.
z-index Z-Index Controls the stacking order of overlapping elements.
overflow Overflow Defines how to handle overflow content in an element (e.g., visible, hidden, scroll, auto).
opacity Opacity Sets the transparency level of an element.
box-shadow Box Shadow Adds shadow effects around an element’s frame.
text-shadow Text Shadow Adds shadow effects to text within an element.
line-height Line Height Specifies the height of a line of text.
cursor Cursor Defines the mouse cursor displayed when hovering over an element.
